# Sound Assets

Both sound sets must use the same cue filenames so the two demos stay mirrored:

```text
sounds/
  basic/
    digit.wav
    enter.wav
    cancel.wav
    start.wav
    mist.wav
    scan.wav
    dock.wav
  functional/
    digit.wav
    enter.wav
    cancel.wav
    start.wav
    mist.wav
    scan.wav
    dock.wav
```

Replace any file with a custom WAV using the same filename to swap the sound without changing the interface.

The current files are rendered placeholders based on the original generated Web Audio tones.
